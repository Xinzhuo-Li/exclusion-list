About This Project

The EMRTS Medicare/Medicaid Provider Exclusion Database initiative aims to make it easier to verify whether a provider appears on a state Medicaid exclusion list. This application reads from a PostgreSQL database populated by an ETL pipeline that normalizes each state's raw files to the OIG LEIE 18-field schema.

New state data can be added through the ETL pipeline without changes to this web application — sources are discovered dynamically from the database.
